# Folder Structure for the project.
 
 ```bash
 ├── Project
 │   ├── Network Gateway
 │   │   ├── Services
 │   │   │   ├── Module's ApiService
 │   │   └── APIClient Base.
 │   ├── Models
 │   │   └── Model classes.
 │   ├── Managers
 │   │   └── Session & User dedault managers.
 │   ├── Fonts
 │   │   └── Custom Fonts for the entire applition. Refer [Apple Docs](https://developer.apple.com/documentation/uikit/text_display_and_fonts/adding_a_custom_font_to_your_app)
 │   ├── Libraries
 │   │   └── External Libraries and Dynamic frameworks goes here.
 │   ├── Extensions
 │   │   └── Extensional of entire application goes here.
 │   ├── Helpers
 │   │   └── Helper class for the entire application goes here.
 │   ├── Modules
 │   │   └── Module
 │   │       └── Module Related Files.(See readme inside readme)
 ├── AppDelegate.swift
 ├── Storybaords
 ├── Assets
 └── plist
 ```
